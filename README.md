This repository contains programs and resources for the **ADA (Analysis and Design of Algorithms) Laboratory**.  
It includes implementations of fundamental algorithms, problem-solving techniques, and performance analysis.

Contents
- Sorting Algorithms (Quick Sort, Merge Sort, Heap Sort, etc.)
- Searching Algorithms (Binary Search, Linear Search)
- Graph Algorithms (DFS, BFS, Dijkstra’s, Kruskal’s, Prim’s)
- Dynamic Programming (Knapsack, Matrix Chain Multiplication, etc.)
- Greedy Algorithms
- Divide and Conquer techniques
